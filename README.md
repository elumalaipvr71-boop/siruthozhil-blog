# Siruthozhil Blog - Vercel Template

This is a simple Vercel blog template that fetches posts automatically from the Blogger site [siruthozhil.in](https://siruthozhil.in) using RSS feed.

## Features
- Mobile friendly design
- SEO optimized basic layout
- Fetches Blogger posts automatically
- Ready to deploy on Vercel
- Domain ready (siruthozhil.in)

## Files
- `index.html` : Homepage layout
- `style.css` : CSS design
- `script.js` : RSS fetch code

## Deployment
1. Upload files to a GitHub repository
2. Connect the repository to Vercel
3. Deploy and optionally connect your domain
